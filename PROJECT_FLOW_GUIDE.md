The app is organized in a clean way. Here's a simple explanation:

| Folder | What It Contains |
|--------|------------------|
| `pages/` | Different screens (Home, Login, Games, etc.) |
| `components/` | Reusable building blocks (Buttons, Cards, etc.) |
| `services/` | Code that talks to the server (database) |
| `context/` | Shared data (like "who is logged in") |
| `layout/` | Page templates (Header, Footer arrangement) |
| `routes/` | URL paths and navigation rules |
| `constants/` | Fixed values (game settings, URLs) |
| `utils/` | Helper functions |
| `types/` | Data structure definitions |
